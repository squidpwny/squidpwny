CORS PoC for Strapi Admin Panel

This page, hosted on a different origin (e.g., `attacker.com`), attempts to fetch the content of `https://cms.bumba.global/admin`. If the target server has a misconfigured CORS policy (e.g., `Access-Control-Allow-Origin: *` and `Access-Control-Allow-Credentials: true`), the response will be displayed below. Otherwise, the browser will block access due to the Same-Origin Policy.

Click the button to perform the fetch request and see the result here.

**Important:** Check your browser's developer console (F12 > Console/Network) for detailed CORS errors.
